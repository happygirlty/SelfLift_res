# SelfLift Project Page

This repository hosts the static project homepage for SelfLift through GitHub Pages.

## Files

- `index.html`: page content and section order.
- `styles.css`: layout and visual styling.
- `assets/selflift_visual.png`: qualitative visual figure.
- `assets/selflift_overview.png`: method overview figure.
- `assets/main_table.png`: quantitative result table.

## Required Image Uploads

Upload the following files into the `assets/` folder:

- `selflift_visual.png`
- `selflift_overview.png`
- `main_table.png`

## GitHub Pages

In repository settings, enable:

```text
Settings -> Pages -> Deploy from a branch -> main -> /root
```

The project URL should be:

```text
https://happygirlty.github.io/SelfLift.github.io/
```
